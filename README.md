# 📊 Analisador AMS - ForceBeat

Sistema profissional de análise de incidentes ServiceNow com IA.

## 🚀 Funcionalidades

- ✅ Análise completa de SLA e TMA
- ✅ 8 abas de visualização
- ✅ Chat IA com Google Gemini 2.5
- ✅ Gráficos interativos
- ✅ Exportação de relatórios

## 📥 Download

Baixe o executável Windows em [Releases](../../releases/latest)

## 🖥️ Requisitos

- Windows 7/8/10/11 (64-bit)
- 4 GB RAM (8 GB recomendado)

## 📖 Como usar

1. Baixe `AnalisadorAMS.exe`
2. Duplo clique
3. Upload planilha ServiceNow
4. Explore análises!
